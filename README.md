# フードスタイリング アシスタント

プロのフードスタイリストのための撮影プラン作成ツールです。

## 機能

- **商品分析**: 商品画像をアップロードすると、中身の形状を分析
- **スタイリング提案**: 形状に合わせた5つのスタイリング案を提案
- **画像生成**: 選んだ案に基づいて完成イメージを生成
- **まとめ出力**: 全商品の撮影プランを表形式でまとめ、買い物リストを作成
- **履歴保存**: 過去のセッションをブラウザに保存

## セットアップ

### 1. 依存関係のインストール

```bash
cd food-styling-assistant
npm install
```

### 2. 環境変数の設定

`.env.local` ファイルをプロジェクトルートに作成し、Gemini API キーを設定：

```
GEMINI_API_KEY=your_gemini_api_key_here
```

API キーは [Google AI Studio](https://aistudio.google.com/app/apikey) から取得できます。

### 3. 開発サーバーの起動

```bash
npm run dev
```

http://localhost:3000 でアプリが起動します。

## Vercel へのデプロイ

### 1. Vercel にプロジェクトをインポート

1. [Vercel](https://vercel.com) にログイン
2. "Add New" → "Project" をクリック
3. GitHubリポジトリを選択、またはフォルダをドラッグ＆ドロップ

### 2. 環境変数の設定

Vercel のプロジェクト設定で以下の環境変数を追加：

- `GEMINI_API_KEY`: Google AI Studio から取得した API キー

### 3. デプロイ

"Deploy" をクリックすると自動でビルド＆デプロイされます。

## 使い方

### 基本フロー

1. **商品を追加**: 商品画像をアップロード、または商品名を入力
2. **分析を確認**: AIが商品の形状を分析
3. **案を選択**: 5つの提案から気に入った案をクリック
4. **画像生成**: 「お願いします」「GO」などと言うと完成イメージを生成
5. **次へ進む**: 「次の商品」または「終わり」と入力
6. **まとめを確認**: 全商品完了後、買い物リストと共にまとめ表を出力

### ショートカット

- `Enter`: メッセージ送信
- `Shift + Enter`: 改行

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **AI**: Gemini 2.0 Flash (テキスト生成 + 画像生成)
- **データ保存**: LocalStorage
- **ホスティング**: Vercel
